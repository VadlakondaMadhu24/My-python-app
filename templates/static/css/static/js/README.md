# ☀️ SUN PHARMACY - Online Medicine Store

A beautiful, responsive pharmacy web application built with Flask, HTML, CSS, and JavaScript.

## 🌟 Features

### 🛍️ Shopping Experience
- Browse 6+ medicines across multiple categories
- Search functionality for quick medicine discovery
- Detailed product information with images
- Real-time stock availability
- Smooth shopping cart with persistent storage

### 🛒 Cart Management
- Add/remove items with quantity control
- Real-time cart updates with badge counter
- Tax calculation (10%)
- Price summary display
- LocalStorage for persistent cart data

### 💳 Checkout System
- Customer information form
- Order confirmation with unique ID
- Professional order receipt
- Email and phone validation

### 🎨 Beautiful UI/UX
- Modern gradient design
- Professional pharmacy images from Unsplash
- Smooth animations and transitions
- Responsive layout (mobile, tablet, desktop)
- Font Awesome icons
- Accessibility features

### 📱 Responsive Design
- Mobile-first approach
- Works on all device sizes
- Touch-friendly buttons
- Optimized for slow connections
